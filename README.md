주제 선정 이유
> 최근 COVID-19로 인하여 비대면 수업이 증가하는 가운데 학교에서 수행되는 재난 대피 요령, 소화기 사용 방법 등 재난 대처 방법을 대면으로 직업 체험해보는 것이 어려워졌다.
> 또한, 영상을 통해 재난 대처 방법을 숙지하는 것보다 게임을 플레이하며 지식을 습득하는 방안이 더 좋을 것 같아서 주제를 선정하였다.
